## Getting Started

### Install pyenv & pipenv

For MacOS:

```
brew install tesseract tesseract-lang
brew install pyenv
pip install --user pipenv
```

### Create the virtual environment

```
pipenv install
```

### Using the virtual environment

Drop into the shell:

```
pipenv shell
```

Or run commands in the shell:

```
pipenv run ./pincidents.py [...]
```
